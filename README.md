# Agent Framework

A practical architecture and implementation blueprint for building single-agent and multi-agent software.

This repository is organized as a documentation-first starter project. It turns one large architecture blueprint into ordered, reusable implementation documents. Start with the quick-start guide, then move through the numbered files in order.

## Project Name

The Chinese idea “agent 框架” is named here as **Agent Framework**. The repository folder is `agent-framework`.

## Suggested First Milestone

Build one reliable single-agent loop that can call tools, recover from failures, persist transcripts, and never send invalid `tool_use` / `tool_result` pairs. Multi-agent behavior comes after that foundation is stable.
